# BattleCity

Порт игры Battle city с консоли NES на Unity engine

Репозиторий является фороком.

Главная цель форка - адаптировать проект для обучения игре нейросети

|   |   |
|---|---|

## Управление

**R** Перезапуск и смена кол-ва игроков

### Игрок 1:

**WASD** Движение.
**G** Стрельба.

### Игрок:

**Стрелки** Движение.
**L** Стрельба.
